Style-Preserving Document Generation

HWPX 원본 구조는 유지하고 내용만 AI로 갈아끼우는 데모

Template-first workflow

핵심 프로세스

원본 문서를 분석하고 스타일과 구조를 가져온 뒤 내용을 치환합니다.

데모는 HWPX를 직접 새로 조립하지 않습니다. 먼저 템플릿 문서를 풀어 XML을 분석하고, 제목과 목차에 맞는 문장만 교체한 뒤 다시 `.hwpx`로 묶습니다.

01 원본 업로드

한글 양식에서 서식과 문단 구조를 읽습니다.

02 AI 제목/목차 입력

사용자가 제목과 개요를 주면 본문 초안을 생성합니다.

03 XML 치환

스타일 ID는 유지하고 텍스트 노드만 바꿉니다.

04 HWPX 재패키징

다시 묶어서 검토용 결과 문서를 만듭니다.
